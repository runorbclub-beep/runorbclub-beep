## 🛠️ Projects

### 📅 Book Chance — Free Booking System
**[→ View on GitHub](https://github.com/runorbclub-beep/book-chance)**

Free, open-source booking system. Build it in 10 minutes. Use it for clients, trade it for services, or sell it. MIT License.

**Two templates included:**

| Template | Best For | Example |
|----------|----------|---------|
| 🧑‍🏫 **Coach** | Personal trainers, tutors, consultants | "Book a session with Coach A" |
| 🏟️ **Venue** | Golf ranges, tennis courts, studios | "Reserve Bay A, 3pm-4pm" |

**What's inside:**
- ✅ User login & booking management
- ✅ Admin panel (check-in, cancel, notifications)
- ✅ Group notifications (who booked, cancelled, no-showed)
- ✅ Deposit system
- ✅ AI-powered generator (zero code needed)
- ✅ One-click deploy to Cloudflare Pages (free)

**Live Demo:** [golf.bazu-ranking.cn](https://golf.bazu-ranking.cn) — a real booking system running for a golf driving range.
